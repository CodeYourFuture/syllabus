How to deploy a server on Render

  1. Make sure you have connected your GitHub account to Render (see How to connect your GitHub to Render)
  2. Log into the Render dashboard

  1. Click on the New + button

  1. Click on Web Service from the dropdown list

  1. Select your repo to connect to Render by clicking on the Connect button

  1. Fill out the Name field, which will be used to label the server within Render

  1. Provide a Start Command field. This should be in the format node YOUR_FILENAME.js, it is likely node server.js

  1. Create your new server by clicking on the Create Web Service button

  1. You should have been taken to the Build page for your new server
  2. Wait for the build to complete (it may take a couple of minutes). It should display "Build successful" as well as "Live"

  1. You can view your newly deployed server by clicking the URL at the top of Web Service page
